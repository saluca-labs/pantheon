# Pantheon Troubleshooting

The canonical self-host troubleshooting reference lives at:

**[`apps/platform-api/deploy/TROUBLESHOOTING.md`](https://github.com/salucallc/pantheon/blob/main/apps/platform-api/deploy/TROUBLESHOOTING.md)**

It covers:

- platform-web won't start (env vars, native modules, BFF connection)
- platform-api startup / migration failures
- SoulAuth federated auth confusion
- `agent.yaml` import errors
- Provider key resolution failures
- Store adapter config issues
- Postgres health

For a flow-chart-style decision tree, see
[`drilldowns/troubleshooting-flowcharts.md`](https://github.com/salucallc/pantheon/blob/main/apps/platform-api/docs/drilldowns/troubleshooting-flowcharts.md).
